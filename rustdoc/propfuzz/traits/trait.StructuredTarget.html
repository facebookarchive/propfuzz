Represents a structured fuzz target.

Functions written with the propfuzz macro are converted to implementations of this trait.

A trait that implements Propfuzz can be used both as a standard property-based test, and as a target for structured, mutation-based fuzzing.

Structured, mutation-based fuzzers use random byte sequences as a pass-through RNG. In other words, the random byte sequences act as something like a DNA for random values.
